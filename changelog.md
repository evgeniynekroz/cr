## [beta-1.0.0] — 2026-04-18

### 🎉 Initial Public Beta

**Added**
- Level submission system via in‑game button (blue star)
- **Sent** list — review queue visible to all players
- **Recent** list — latest rated levels
- **Random** list — discover random rated levels
- **Top Likes** — live ranking by GD like counts (fetches data from GD servers)
- Custom difficulty icons on rated level pages
- "Rated by" label on rated level pages
- Full moderator panel — rate, set difficulty, delete ratings
- Admin list loaded from database (no hardcoded names)
- Cancel tokens — no crashes when closing screens mid‑request
- Rate info cache — instant repeated lookups

**Known Issues**
- Navigation arrows may be reversed on some devices (visual only, functionality works)
- Top Likes list may load slowly due to GD server rate limits
- Level names sometimes fall back to "ID xxxxx" if GD servers are unresponsive
