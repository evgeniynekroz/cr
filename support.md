## ❓ Common Issues

**Q: The mod doesn't appear in the Geode menu**  
> Make sure you placed the `.geode` file in the correct `geode/mods` folder and restarted GD.

**Q: I see "Failed to load mod"**  
> Ensure you have Geode v5.6.0 or newer. Try reinstalling the mod.

**Q: The custom rate button doesn't show on level pages**  
> The button only appears on unrated levels (official stars = 0). Also check if another mod conflicts.

**Q: I'm a moderator but the Delete button is missing**  
> The moderator list is managed in the database. Contact the developer to be added.
